# dashboard-gestao-operacional
Dashboard desenvolvido em Power BI para análise de indicadores de gestão operacional, com foco em acompanhamento de performance, monitoramento de KPIs e apoio à tomada de decisão, utilizando dados fictícios.

<video src="Painel%20TMA%20-ativos/dashboard.gif%20GIF.mp4" width="100%" controls></video>

🎯 Objetivo

Desenvolver meu primeiro dashboard focado em indicadores operacionais para acompanhamento de performance.

📌 Problema

Sem indicadores claros, a gestão operacional se torna reativa e pouco eficiente.

📈 Análises desenvolvidas

Indicadores de desempenho (KPIs)

Acompanhamento operacional

Visão geral da performance

🛠️ Ferramentas utilizadas

Excel

📊 Insights possíveis

Monitoramento contínuo da operação

Apoio na tomada de decisão estratégica

⚠️ Observação

Projeto fictício desenvolvido para estudo e prática, sendo este o meu primeiro dashboard.
